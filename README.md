# Blinkit-Sales-Analysis-Dashboard
The dataset used for this project contains information related to retail sales transactions across different Blinkit outlets.

📌 Project Overview

This project presents an interactive *Power BI dashboard* developed to analyze sales performance for Blinkit retail outlets. The dashboard provides insights into total sales, product categories, outlet characteristics, and customer ratings.

The objective of this analysis is to understand sales distribution across different outlet types, item categories, outlet sizes, and locations, while identifying key factors that influence overall business performance.

The dashboard enables decision-makers to monitor performance metrics and gain insights through interactive visualizations.

Key Variables in the Dataset

- Total Sales – Total revenue generated from product sales
- Average Sales – Average sales per item
- Number of Items – Total number of products sold
- Average Rating – Customer rating for products or outlets
- Item Type – Product categories such as fruits, snacks, dairy, frozen foods, etc.
- Fat Content – Product classification (Low Fat / Regular)
- Outlet Establishment Year – Year the outlet was established
- Outlet Size – Small, Medium, or High outlet size classification
- Outlet Location Type – Tier 1, Tier 2, or Tier 3 city classification
- Outlet Type – Grocery store or supermarket category
- Item Visibility – Product visibility or shelf presence

Key Visualizations

1. Sales Trend by Outlet Establishment
- Shows how sales vary based on outlet establishment year.

2. Sales by Item Type
- Identifies which product categories generate the highest revenue.

3. Fat Content Distribution
- Compares sales between low-fat and regular products.

4. Outlet Size Analysis
- Displays sales contribution from small, medium, and large outlets.

5. Outlet Location Analysis
- Compares sales performance across Tier 1, Tier 2, and Tier 3 locations.

6. Outlet Type Performance
- Compares grocery stores and supermarkets using metrics such as:
  - Total sales
  - Number of items
  - Average sales
  - Customer ratings
  - Item visibility

🔍 Key Insights

- Tier 3 locations contribute the highest sales among outlet locations.
- Regular fat products generate slightly higher revenue compared to low-fat products.
- Supermarkets sell a higher number of items compared to grocery stores.
- Medium and large outlet sizes contribute significantly to total revenue.
- Product categories such as fruits, snacks, and household items are among the top-performing segments.
